**Note: Please use Brackets or any ide that creates a server to run the application.


Or follow the following process

Step 1: Open Command Prompt or GitBash or any CommandLine Application you prefer
Step 2: Navigate to the project directory: in this case it will be CodingChallenge
Step 3: Type the following then press enter
        npm start
Step 4: npm will install all the dependencies and will start a http-server. After starting it will show  something like:
        Available on:
        http://localhost:8000
Step 5: Use this address to access the application.
